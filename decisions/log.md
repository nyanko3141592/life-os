# 意思決定ログ

重要な決断を記録するファイルです。
「なぜその決断をしたか」を残しておくと、将来の自分が助かります。

フォーマット:

```
## YYYY-MM-DD: [決断の内容]

**背景:**

**選択肢:**
- A:
- B:
- C:

**選んだこと:**

**理由:**

**懸念点:**

**振り返り予定:** YYYY-MM-DD
```
